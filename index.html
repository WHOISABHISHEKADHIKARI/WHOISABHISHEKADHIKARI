About this repo

This is a lightweight HTML boilerplate for the gigs project. Use it as the landing page or starter UI for your repo.

Accessible

Semantic markup, clear headings and ARIA-friendly structure.

SEO-ready

Meta tags, OG tags and JSON-LD included.

Easy to extend

Small CSS footprint and simple components to customize quickly.

Quick start

  1. Copy this file to index.html in your repo root.
  2. Replace placeholders: title, description, og-image, author, and repo URL.
  3. Add assets to /assets (favicon, og-image). Optionally add a manifest.json.
  4. Add your styles or integrate Tailwind / Bootstrap as needed.
